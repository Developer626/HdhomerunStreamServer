HdhomerunStreamServer

Stream from HDhomerun to other devices

Windows

What is it?

HdhomerunStreamServer allows you to stream from a HDhomerun device on the network to an Android device on the same network. HdhomerunStreamServer uses VLC to transcode the stream so that HD channels are playable on your Android device.

How to use it?

First you'll need VLC installed on the computer that you'll be running HdhomerunStreamServer on. You'll also need an Android device with the HomerunTV app installed on the same network as the PC.

Setup
